# Visual style map

| Meaning | PyMOL colour family |
|---|---|
| hydrophobic base | `tv_green` |
| neutral base | `sand` |
| hydrophilic base | `marine` |
| unknown residue base | `gray70` |
| charged review | `orange` |
| polar contextual review | `yellow` |
| ligand | `magenta` |
| ligand shell | `cyan` |
| membrane boundaries | translucent blue/orange |

Hydropathy is applied first. Ligand context follows, and review styles are re-applied last so
higher-priority residues remain visible. The palette still requires interactive accessibility
review.
